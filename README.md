# Node Word Count

Count total words sum in text files.

## Features

- No dependencies
- Only Node.js API used
- Uses ES module import
- Example text file included in repo (46 KB file size)

- Node.js v18.13.0
- npm 8.19.3

## Usage

Run in console: `npm start`
